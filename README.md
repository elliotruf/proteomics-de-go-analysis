# proteomics-de-go-analysis
Exploratory analysis (1 test sample, 1 control) of total proteome and phosphoproteome datasets performed as part of my MSc in Applied Bioinformatics & Genomics. 

The workflow includes:
- data preprocessing
- fold-change-based expression analysis
- Gene Ontology enrichment
- data visualisation

Includes an optional sensitivity analysis setting, allowing for parallel analyses to be conducted at differing thresholds.

Tools used: R, tidyverse, janitor, clusterProfiler, pheatmap, & ggplot2.
